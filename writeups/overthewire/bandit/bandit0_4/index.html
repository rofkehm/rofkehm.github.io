CTF Writeup: Bandit - Niveles 0 a 4 (OverTheWire)

Bandit es un wargame de OverTheWire enfocado en fundamentos de Linux, manejo de archivos, nombres especiales y permisos. Cada nivel requiere encontrar la contraseña del siguiente usuario mediante comandos básicos.

Metodología y Solución

Nivel 0 → 1

Conexión inicial: 

voodoo@voodoo-pc $ ssh bandit0@bandit.labs.overthewire.org -p 2220

bandit0@bandit:~$ ls

Contraseña nivel 1: ZjLjTmM6FvvyRnrb2rfNWOZOTa6ip5If

Nivel 1 → 2

Conexión con la contraseña anterior: 

voodoo@voodoo-pc $ ssh bandit1@bandit.labs.overthewire.org -p 2220
bandit1@bandit.labs.overthewire.org's password: ZjLjTmM6FvvyRnrb2rfNWOZOTa6ip5If

Archivo con nombre - (guión). No podemos usar cat - (se interpreta como flag). Usamos ./-: 

bandit1@bandit:~$ ls
-
bandit1@bandit:~$ cat ./-
263JGJPfgU6LtdEvgfWU1XP5yac29mFx

Contraseña nivel 2: 263JGJPfgU6LtdEvgfWU1XP5yac29mFx

Nivel 2 → 3

Archivo con espacios y guiones: 

voodoo@voodoo-pc $ ssh bandit2@bandit.labs.overthewire.org -p 2220
bandit2@bandit:~$ ls
--spaces in this filename--
bandit2@bandit:~$ cat ./--spaces\ in\ this\ filename--
MNk8KNH3Usiio41PRUEoDFPqfxLPlSmx

Usamos \ para escapar espacios o autocompletado con Tab.

Contraseña nivel 3: MNk8KNH3Usiio41PRUEoDFPqfxLPlSmx

Nivel 3 → 4

Carpeta inhere con archivo oculto: 

voodoo@voodoo-pc $ ssh bandit3@bandit.labs.overthewire.org -p 2220
bandit3@bandit:~$ ls
inhere
bandit3@bandit:~$ cd inhere/
bandit3@bandit:~/inhere$ ls
bandit3@bandit:~/inhere$ ls -a
.  ..  ...Hiding-From-You
bandit3@bandit:~/inhere$ cat ...Hiding-From-You
2WmrDFRmJIq3IPxneAaMGhap0pFhF3NJ

ls -a muestra archivos ocultos (inician con .).

Contraseña nivel 4: 2WmrDFRmJIq3IPxneAaMGhap0pFhF3NJ

Comandos Clave Aprendidos

  • ssh user@host -p 2220 – Conexión remota
  • ls, cat, cd – Navegación y lectura
  • ./archivo – Leer archivo con nombre especial
  • ls -a – Mostrar archivos ocultos
  • \ o Tab – Escapar espacios

Recursos Utilizados

  • Bandit - OverTheWire
  • man ls

Notas

  • Las contraseñas son dinámicas y cambian con el tiempo.
  • Ideal para aprender manejo de archivos y nombres especiales en Linux.
